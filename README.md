# Spotify Api Test

A playground project template for training purposes on the Spotify web api.

## Installation

``` bash
yarn install
```

### Requirements

[Node.js LTS](https://nodejs.org/)

## Usage

### Running the tests

``` bash
yarn test
```

### eslint fix

``` bash
yarn lint
```
